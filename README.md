# 號碼抽籤器

一個用於課堂點名、分組、Q&A 抽人等情境的網頁抽籤工具。  
支援範圍設定、排除、重複/不重複模式、多次抽取、顏色標記次數等功能。

---

## 🎯 功能特色

- 設定號碼範圍 **1–100**
- 可排除特定號碼（例如請假、缺席等）
- **重複模式** / **不重複模式**
- 支援一次抽 **1–20 位**
- 具有麻仔台風格 **抽籤動畫**
- 顯示抽中號碼次數（最多 10 次）
- 超過 10 次後自動排除不再抽取
- 範圍變化時自動調整 **每排顯示格數**
- 顏色依次數顯示：  
  `1紅 → 2橙 → 3黃 → 4綠 → 5藍 → 6靛 → 7紫 → 8白 → 9灰 → 10黑`
- 重設不會清除範圍與排除條件

## 🕹 使用方式

### 1. 設定抽籤條件

可設定：

- **範圍：** `1 ~ N`
- **排除：** `3, 7, 12`
- **抽籤模式：**
  - 重複模式（允許同一人多次被抽）
  - 不重複模式（抽過的不再抽）
- **抽取數量：** `1–20`
- **動畫啟用/關閉**

### 2. 抽籤

按下 `開始抽籤` → 若啟動動畫會跑 slot machine 效果。

### 3. 統計顏色

| 次數 | 顏色 |
|---|---|
| 1 | 🔴 紅 |
| 2 | 🟠 橙 |
| 3 | 🟡 黃 |
| 4 | 🟢 綠 |
| 5 | 🔵 藍 |
| 6 | 🟣 靛 |
| 7 | 🟪 紫 |
| 8 | ⚪ 白 |
| 9 | ⚫ 灰 |
| 10 | ⬛ 黑 |

第 10 次會顯示，但之後不再抽。

---

## ♻ 重置規則

按下 `重置抽籤紀錄`：

- **會清除：抽籤次數**
- **不會清除：範圍與排除設定**
- 切換重複/不重複模式時也視同重置

## 🛠 部署方式

可直接放 `Vercel` 或任何靜態 hosting，例如：

- GitHub Pages  
- Netlify  
- Firebase Hosting  
- Nginx 靜態伺服器

部署至 Vercel 步驟：

```
git init
git add .
git commit -m "feat: initial"
git remote add origin <repo>
git push -u origin main
```

Vercel → Import Git repo → Deploy 即完成 CI/CD。

  - Exclude

  - Hard cap = 10 次
